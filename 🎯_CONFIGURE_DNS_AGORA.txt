═══════════════════════════════════════════════════════════════════
  PASSO 2: SELECIONAR DOMÍNIO
═══════════════════════════════════════════════════════════════════

  1. Clique em "diotec360.com"
  2. Vá em "DNS" no menu lateral


═══════════════════════════════════════════════════════════════════
  PASSO 3: ADICIONAR REGISTRO
═══════════════════════════════════════════════════════════════════

  Clique em "Add Record" e preencha:

  ┌─────────────────────────────────────────────────────────────┐
  │                                                             │
  │  Type:  CNAME                                               │
  │  Name:  hf                                                  │
  │  Value: diotec-aethel-judge.hf.space                        │
  │  TTL:   60                                                  │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘

  Clique em "Save"


═══════════════════════════════════════════════════════════════════
  PASSO 4: AGUARDAR PROPAGAÇÃO
═══════════════════════════════════════════════════════════════════

  ⏱️  Aguarde 2-5 minutos para o DNS propagar


═══════════════════════════════════════════════════════════════════
  PASSO 5: TESTAR
═══════════════════════════════════════════════════════════════════

  Execute no terminal:

  curl https://hf.diotec360.com/health

  Deve retornar:
  {"status":"healthy","version":"3.0.5"}
