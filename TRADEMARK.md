# Trademark Usage Policy

**Effective Date**: February 19, 2026  
**Owner**: DIOTEC 360

## Overview

This document outlines the trademark usage policy for "Aethel," "DIOTEC 360," and related marks. The Aethel Protocol is open source under the Apache 2.0 license, but the trademarks remain the exclusive property of DIOTEC 360.

## Purpose

The purpose of this policy is to:
- Protect the integrity and reputation of the Aethel brand
- Ensure users can identify official Aethel software and services
- Allow fair use of trademarks for legitimate purposes
- Prevent confusion about the source and quality of Aethel-related products

## Permitted Uses

You MAY use the Aethel trademarks without explicit permission for:

### 1. Referential Use
- Describing that your software is compatible with Aethel
- Stating that your product integrates with or uses Aethel
- Referring to the Aethel Protocol in technical documentation
- Writing articles, blog posts, or academic papers about Aethel

**Examples**:
- ✓ "Our banking platform uses Aethel Protocol for transaction verification"
- ✓ "This library provides Python bindings for Aethel"
- ✓ "Compatible with Diotec360 v1.9.0 and later"

### 2. Community Use
- Creating user groups or meetups (e.g., "Boston Aethel Users Group")
- Organizing conferences or events about Aethel (with clear indication of unofficial status)
- Creating educational content and tutorials

**Requirements**:
- Must clearly indicate the group/event is not officially endorsed by DIOTEC 360
- Must not imply official affiliation or sponsorship
- Must include disclaimer: "This is an independent community initiative, not officially affiliated with DIOTEC 360"

### 3. Open Source Distribution
- Distributing unmodified Aethel Protocol source code
- Creating packages for package managers (pip, npm, etc.)
- Mirroring the official repository

**Requirements**:
- Must not modify the source code in ways that could harm the Aethel reputation
- Must clearly indicate if the distribution is unofficial
- Must preserve all copyright and trademark notices

### 4. Fair Use
- News reporting about Aethel
- Academic research and citations
- Comparative advertising (truthful and non-disparaging)
- Parody and commentary (clearly marked as such)

## Prohibited Uses

You MAY NOT use the Aethel trademarks for:

### 1. Confusing or Misleading Use
- ✗ Implying your product is the official Aethel Protocol
- ✗ Using "Aethel" as part of your product name without permission
- ✗ Creating domain names that could be confused with official Aethel sites
- ✗ Suggesting official endorsement or sponsorship by DIOTEC 360

**Examples of prohibited names**:
- ✗ "Aethel Pro"
- ✗ "Aethel Enterprise"
- ✗ "Aethel Cloud"
- ✗ "AethelDB"

### 2. Modified Versions
- ✗ Distributing modified versions of Aethel under the "Aethel" name
- ✗ Creating forks and calling them "Aethel [Something]"

**If you create a modified version**:
- You MUST use a different name that clearly distinguishes it from Aethel
- You MUST clearly state it is derived from Aethel Protocol
- You MUST NOT imply it is the official Aethel implementation

**Example**:
- ✓ "FinanceChain Protocol (derived from Aethel Protocol)"
- ✗ "Aethel FinanceChain"

### 3. Commercial Services
- ✗ Offering "Aethel hosting" or "Aethel as a Service" without permission
- ✗ Using "Aethel" in your company name
- ✗ Selling "Aethel training" or "Aethel certification" without authorization

**Note**: If you want to offer commercial services using Aethel, contact DIOTEC 360 for partnership opportunities.

### 4. Quality Degradation
- ✗ Using Aethel marks in connection with low-quality or malicious software
- ✗ Associating Aethel with illegal activities
- ✗ Using marks in a way that damages the Aethel reputation

## Logo Usage

If DIOTEC 360 provides an official Aethel logo:

### Permitted
- Using the logo to indicate Aethel compatibility
- Including the logo in presentations about Aethel
- Displaying the logo on websites that use Aethel

### Requirements
- Must not modify the logo (colors, proportions, elements)
- Must maintain clear space around the logo
- Must not use the logo as the primary branding for your product
- Must link to official Aethel website when used online

### Prohibited

- Using the logo in a way that implies endorsement
- Making the logo the most prominent element of your branding
